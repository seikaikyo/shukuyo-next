C6禁止事項

ID禁止原因正確做法
C6.1Hardcode hex/rgb無法跟隨主題切換使用 CSS 變數 / Tailwind class

C6.3四象名（青龍/朱雀/白虎/玄武）二十八宿系統，非原典只用序號 + 七曜名
C6.4幸運方位/色/數五行自創對應表，無原典依據使用三九秘法宜忌（T1299）
C6.5事業/感情/健康/財運分類現代編輯，非原典使用三九日型宜忌
C6.6「平」「小吉」「末吉」等級後端只有四等級只用大吉/吉/小凶/凶
C6.7六害宿非原典，已移除不使用
C6.8Emoji開發規範禁止使用 lucide-react icon 或漢字
C6.9Badge bg 不透明與卡片底色衝突一律 12% 透明度
